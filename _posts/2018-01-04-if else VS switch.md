## compare

- 5개 이하 비교

	- 차이 무

- 5개 이상 비교

	- switch

		- 해쉬 테이블 룩업 테이블을 구현해서 아이템에 접근
		- 어떤 아이템이든지 접손 시간 같음 => O(1)
	
	- if else

		- 조건에 맞을 때까지 비교
		- 최대 아이템 갯수만큼의 비교 시간이 걸릴 수 있음 => O(N)
