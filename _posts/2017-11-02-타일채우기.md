## 풀이

1. d[i] 

	- 3 * i 타일을 채우는 방법의 수 

2. 가장 마지막에 올 수 있는 경우를 찾아야 한다.

	![img](../img/1.jpg)
	
	```
	d[i] = 3 * d[i-2]
	```
	
	![img](../img/2.jpg)
	
	```
	d[i] = 2 * d[i-4]
	```
	
	![img](../img/3.jpg)
	
	```
	d[i] = 2 * d[i-6]
	```

3. d[i]

	```
	3*d[i-2] + 2*d[i-4] + 2*d[i-6] + ... + 2*d[0]
	```
	어차피, 가로 길이가 홀수이면 타일로 채울 수 X
