#  하나의 원격 저장소를 두 개의 지역 저장소가 동기화하는 방법

````
Git clone 주소 git_home	 //그럼 git_home이라는 곳이 생김
Git clone 주소 git_office	 //그럼 git_office이라는 곳이 생김


- git_home 디렉토리에서
Vim f1.txt	 	//내용   a b c
Git commit -am 3
Git push


- git_office 디렉토리에서
Git pull 
vim f1.txt		//내용 a b c 에 d 추가
Git commit -am ‘4’
Git push


git_home 디렉토리에서
Git pull

```
