11055:가장 큰 증가 부분 수열

Sep 30, 2017 in algorithm

https://www.acmicpc.net/problem/11055

틀린 풀이

  1. d[i]
    • a[1] ~ a[i] 중
    • a[i]를 포함하며,
    • 수열의 합

  2. i번째를 기준으로 i-1번째 ~ 1번째로 하나 하나 검색

  3. i번째보다 작으면 d[i]에 더함

맞는 풀이

  1. 앞 문제와 동일하다

  2. 앞 문제의 조건

    • i < j => a[i] < a[j]
    • max(d[j]) + 1 => 여기서 1은 길이
  3. 1을 점수로 바꿔주면 됨
  4. 햇갈리니까 표를 그리자!!
